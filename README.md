## Install

```
git clone https://github.com/k-nasa/isucon_secret_sauce.git
chmod -R +x  ./isucon_secret_sauce/
cd isucon_secret_sauce

# apt-getを使用してる箇所があるので適宜yumなどに変更する
./setup.sh
```
